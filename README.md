# LAMPSolar
This repo contains the python simulation files for the simple, non-location specific global insolation model developed by Dr. Paul Leu and Sooraj Sharma at the University of Pittsburgh in 2021.

Publication Link: https://www.spiedigitallibrary.org/journals/journal-of-photonics-for-energy/volume-11/issue-4/045501/Solar-module-orientation-and-tracking-type-performance-and-optimization/10.1117/1.JPE.11.045501.short?SSO=1&tab=ArticleLink

This model is based on the Liu-Jordan approximation assuming isotropic diffuse radiation (albedo not considered), and asseses the efficacy of panels at fixed-tilt, 1D- and 2D- tracking configurations.
